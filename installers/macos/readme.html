What will be installed

  • /usr/local/bin/rpc-server - llama.cpp RPC server
  • /usr/local/bin/rpc-worker-beacon - Network discovery beacon
  • LaunchDaemon plists for automatic startup

Ports used

  • TCP 50052 - RPC server (for inference requests)
  • UDP 5005 - Discovery beacon (for auto-detection)

After installation

The services will start automatically. To verify:

sudo launchctl list | grep rpccluster

Uninstallation

To remove RPC Cluster Worker, run:

sudo launchctl bootout system/com.rpccluster.rpcserver
sudo launchctl bootout system/com.rpccluster.beacon
sudo rm /usr/local/bin/rpc-server
sudo rm /usr/local/bin/rpc-worker-beacon
sudo rm /Library/LaunchDaemons/com.rpccluster.*.plist
